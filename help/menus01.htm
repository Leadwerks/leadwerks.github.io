File Menu

New

This will open a new blank map file.

Open...        Ctrl+O

This will bring up a browse window from which you can load maps. The file type will default to .3dw (3D world). If you wish to load another file format, use the Import menu item.

Merge...

This will bring up a browse window from which you can load maps. The loaded map will be merged with any geometry already loaded in the editor.

Save        Ctrl+S

This will save the current loaded map.

Save As...

This will bring up a Save As… dialog box and allow you to save the loaded map with a different name. You can save a .3dw (3D world) or .pfb (prefab) file.

Save Selected...        Ctrl+S

This will bring up a Save As… dialog box and allow you to save the currently selected items in the loaded map. You can save a .3dw (3D world) or .pfb (prefab) file.

Import...

This will bring up a Open… dialog box and allow you to load other file formats, like .map.

Export...

This will bring up a Export… dialog box and allow you to save the loaded map to many file formats, like .x or .map.

Run Map......        F5

Opens the run map dialog, allowing you to save and run the map in an external viewer.

Exit

This command will close 3D World Studio and return to the Windows desktop.
