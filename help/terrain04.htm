Texturing Terrain

Press the terrain editing toolbar button. This will open the terrain editing window.

To apply a texture to a terrain, choose the texture you want from the texture category and texture lists, located in the side panel.

Press the apply texture toolbar button to paint the terrain with the current texture.

You can create additional terrain layers, each with a different texture. To create a new terrain layer, first select a new texture from the texture category and texture lists, located in the side panel. Next, select the layers tab and press the new button.

The new terrain layer will not be visible until it is painted on. To paint a terrain layer, select the paint terrain tool. Click and drag the left mouse button on the terrain in the Perspective or Top view.

If you hold the control key while using the paint terrain tool, the current terrain layer will be erased.

To apply a texture to a terrain layer, choose the texture you want from the texture category and texture lists, located in the side panel. Select the layer tab and select the terrain layer you want to change from the terrain layer list. Press the apply texture toolbar button to paint the terrain layer with the current texture.

You can load a terrain layer from a file. Make sure the layer tab is selected and press the load button. Choose an alphamap image to load. When the image is loaded, the pixel brightness of each pixel will be used to set the terrain layer opacity.
