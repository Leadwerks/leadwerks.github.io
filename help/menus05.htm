Help Menu

Help Contents        F1

Displays the 3D World Studio User's Guide.

Online Resources

This submenu contains useful links to online resources.

Key Shortcuts        F2

This will open the key shortcuts dialog, showing all menu shortcuts the program uses.

Hardware Information...

This will open the hardware information dialog, showing information about your graphics card and supported features and extensions.

Plugin Information...

This will open the plugin information dialog, showing information on all loaded editor plugins.

About...

This will display the about dialog, showing information about the program.
