Adding New Textures

You can add new textures to the texture path directory. By default, this is set to root 3D World Studio directory. The next time you start the program, the textures will be available to choose from.

You can also set the texture path directory to the location where your project's textures are stored. Select the Tools>Options menu item, and change the texture path to your project's texture directory. Textures will then be loaded directly from your project's texture directory.
