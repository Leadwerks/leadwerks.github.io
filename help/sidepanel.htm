Side Panel

The side panel contains dialogs for selecting textures, selecting models, editing visgroups, and editing solid face properties,

Objects

You can select a model from the object category and object lists. You can choose from solids, models, entities, and prefabs. The current object determines how a new object is drawn and what kind of object will be created.

Textures

You can select a texture from the texture category and texture lists. The current texture will be shown in the texture preview window. Press the apply button to apply the current texture to the selected object(s). Press the browse button to open the texture browser. Press the replace button to open the replace texture dialog.

Face Properties

When in select subobject mode the face properties panel will be displayed in the side panel. This allows you to adjust face texture mapping properties.

VisGroups

The visgroups panel allows you to adjust visgroup properties. Unchecking a visgroup in the visgroup list will hide it. The apply button will make the selected object(s) a member of the selected visgroup. The edit button will open the visgroups dialog. The select button will select all objects that are members of the selected visgroup. The purge button will delete any visgroups that do not contain any members.

Use visgroups this to separate your map into separate areas so that you can work on one section at a time.
