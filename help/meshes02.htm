Creating Models

Select the model you want from the object category and object list, located on the side panel.

Click the left mouse button in any of the 2-dimensional viewports. A position marker will appear. You can move this around by clicking and dragging the left mouse button.

Press the right mouse button and select the Create menu. Alternatively, you may press the Enter key.

A model will be created, in the position you placed the marker. It may take a moment to the first time a model is created, because the model data has to be loaded for the first time. Subsequent instances of the model will be created much quicker.
