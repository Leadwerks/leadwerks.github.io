Toolbar

New
This will open a new blank map file.

Open...        Ctrl+O
This will bring up a browse window from which you can load maps. The file type will default to .3dw (3D world). If you wish to load another file format, use the Import menu item.

Save        Ctrl+S
This will save the current loaded map.

Cut        Ctrl+X
Deletes the currently selected objects from the map, and stores a copy of them in memory to be pasted later.

Copy        Ctrl+C
Copies the currently selected objects into memory without removing the original.

Paste        Ctrl+V
Pastes a copy of any cut or copied objects into the current map. Pasted objects will appear in the exact same position as the originals, but will be selected so that you can move them wherever you like.

Wireframe
Switches the current viewport to wireframe render mode.

Solid
Switches the current viewport to solid render mode.

Textured
Switches the current viewport to textured render mode.

Lighting
Switches the current viewport to lighting render mode.

Textured+Lighting
Switches the current viewport to textured+lighting render mode.

Vis Group
Switches the current viewport to visgroup render mode.

Perspective
Switches the current viewport to perspective render view.

Front - XY
Switches the current viewport to front - XY render view.

Top - XZ
Switches the current viewport to top - XZ render view.

Side - ZY
Switches the current viewport to side - ZY render view.

Select Object        Z
Switches to select object mode.

Select Subobject        X
Switches to select subobject mode.

Select Object        C
Switches to select vertex mode.

Rotate X
Rotates the selected object(s) 90 degrees around the X axis.

Rotate Y
Rotates the selected object(s) 90 degrees around the Y axis.

Rotate Z
Rotates the selected object(s) 90 degrees around the Z axis.

Group        Ctrl+G
This will group all selected objects together. If you select any object in the group, the entire group will be selected.

Ungroup        Ctrl+U
This will ungroup any selected grouped objects.

Carve
Performs a CSG carve operation. Any selected solids will carve their volume out of any overlapping unselected solids.

Hollow
Opens the hollow dialog to perform a CSG hollow operation. Any selected solids will be hollowed by the thickness indicated in the Hollow dialog. If a negative thickness is given, the solid will be hollowed outwards.

Slice Plane
Switches to slice plane mode. You can draw a line across any solids. If no objects are selected, any solids the line intersects will be sliced. If any object is selected, only selected solids will be sliced.

Extrude Face
Opens the extrude face dialog. Any solid faces selected will be extruded by the amount indicated in the extrude face dialog. If the editor is in object selection mode all faces of the selected solid(s) will be extruded.

Mirror        Ctrl+M
Opens the mirror dialog, allowing you to mirror all selected objects on any axis.

Get Texture
Switches to get texture mode, allowing you to select any texture in the scene. The mouse cursor will change to a dropper icon, and the texture on any object you click on will be made the current texture.

Apply Texture        Ctrl+A
Applies the current texture to all selected objects, faces, and subobjects in the scene.

Texture Browser        Ctrl+T
Opens the texture browser, allowing you to view and select textures.

Texture Lock
When enabled, texture lock will lock solids' texture mapping in place when they are moved or rotated.

Hide Mask
When in select subobject mode, this will hide the red selection mask shown on selected faces and subobjects.

Model Browser        Shift+M
Opens the model browser, allowing you to view and select models.

Terrain Editing
Opens the terrain editing dialog, allowing you to create and modify terrain geometry.

Render Lights
Opens the render lights dialog, allowing you to calculate lighting. You must have one viewport in lighting or textured+lighting render mode to view map lighting.

Options
Opens the options dialog, allowing you to adjust editor options like video settings and interface colors.

Help Contents
Displays the 3D World Studio User's Guide.
