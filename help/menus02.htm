Edit Menu

Undo        Ctrl+Z

This will allow you to undo the last change you made to the loaded map.

Cut        Ctrl+X

Deletes the currently selected objects from the map, and stores a copy of them in memory to be pasted later.

Copy        Ctrl+C

Copies the currently selected objects into memory without removing the original.

Paste        Ctrl+V

Pastes a copy of any cut or copied objects into the current map. Pasted objects will appear in the exact same position as the originals, but will be selected so that you can move them wherever you like.

Delete        Del

This will delete the currently selected objects from the map, without copying them to memory. When in subobject selection mode, this command will hide any selected solid faces. Hidden faces can still be selected.

Select All

This will select all visible objects in the map.

Select None

This will deselect all objects in the map.

Invert Selection

This will invert the selection state of all visible objects in the map. If an object was selected, this operation will deselect it. If an object wasn't selected, this operation will select it.

Select Object        Z

This will switch the editor to select object mode, allowing you to select whole objects, or groups of objects. This is the default selection mode when the program starts.

Select Subobject        X

This will switch the editor to select suboject mode, allowing you to select solid faces and model subobjects. Select suboject mode is used mostly for applying and adjusting textures.

Select Vertex        C

This will switch the editor to select vertex mode, allowing you to select solid vertices and move them. Model vertices cannot be moved.

Properties...        Shift+Enter

This will open the properties dialog, allowing you to edit properties for the selected object(s).
