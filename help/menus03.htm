View Menu

Toolbar

This will show and hide the editor toolbar.

Side panel

This will show and hide the editor side panel.

Scrollbars

This will show and hide the viewport scrollbars.

Program Log        F3

This will show and hide the program log window.

Map Statistics        F4

This will show and hide the map statistics window.

Increase Grid Size        ]

This will double the grid spacing of the current viewport.

Decrease Grid Size        [

This will half the grid spacing of the current viewport.

Zoom In        +

This will zoom in the current viewport.

Zoom Out        -

This will zoom out the current viewport.

Single Viewport

This will make the current viewport fill the editor viewport area, and hide all other viewports. Selecting it again will switch back to four viewport mode.

Render Mode

You can select the current viewport render mode from this menu. Available render modes are wireframe, solid, textured, lighting, textured+lighting, and vis group.

Render View

You can select the current viewport render view from this menu. Available render views are Perspective, Front X/Y, Top X/Z, and Side Z/Y.

Render Elements

You can select object types to show or hide from this menu. Available object types are solids, models, entities, and terrain.

Render to File...

This will open the Render dialog, allowing you to save any viewport view to an image file.
