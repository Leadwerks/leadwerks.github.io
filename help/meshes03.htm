Modifying Models

Once a model is created, it can be be moved, scaled, and rotated.

Moving Models

To move one or more selected brushes, move the mouse inside the selection bounding box. The mouse icon will change into a four-way arrow.

Press the left mouse button, hold, and drag the mouse.

Release the left mouse button and the selected brushes will be moved to the new position.

Scaling Models

To change a model's scale, select the Edit>Properties menu item. The properties dialog will open.

Select the matrix tab. You can edit the model scale, as well as the position and rotation values.

Press apply or close the properties dialog to enact your changes.

Rotating Models

Brushes can be rotated four ways. The first method uses the rotation mouse tool. The second method uses the rotatation dialog to rotate the selection by an arbitrary value. The third method uses the rotate toolbar buttons to perform a quick 90-degree rotation around any axis. Finally, you can adjust the model matrix directly, by editing the model properties.

Click the left mouse button once within the selection bounding box of a brush. The mouse tool mode will be changed to rotation.

Click the left mouse button on any of the corner selection tabs and drag to rotate the selection.

The rotation mouse tool will lock the rotation angle to 5 degree increments. If you want to rotate the selection by an arbitrary value, select the Tools>Rotate menu item. The rotation dialog will appear. You can input the angles you wish to rotate the selection by, and press Apply to enact the rotation.

You can perform a quick 90-degree rotation around any axis by pressing the Rotate X , Rotate Y , or Rotate Z toolbar buttons.

Finally, you can edit the rotation values directly by selecting the Edit>Properties menu item. The properties dialog will open. Select the matrix tab. You can edit the model rotation, as well as the position and scale values.

Mirroring Models

Model positions can be mirrored by pressing the Mirror toolbar button. However, this will not affect the model's vertices. This is because scaling a model by a negative factor (mirroring) would flip the model triangles, effectively turning it inside-out.
