Special Entities

Some entities have special behavior in the editor. Light entities are recognized by the editor, and used to render lighting in the scene. The editor recognizes both its own 3D World Studio light classes, as well as Torque Engine light classes.

If you wish to use Torque entity classes, you must specify the Torque entity definitions file ("torque.def") in the options dialog.

Point Lights
Point lights emit light in all directions.

3D World Studio point light class:
Classname light
color - light color
intensity - light intensity
range - light range
linearfalloff - indicate linear falloff, or inverse square falloff

Torque Engine point light class:
Classname light_omni
color - light color
intensity - light intensity
falloff2 - light range

Spotlights
A spotlight emits a cone of light. These are useful for simulating artificial light sources.

3D World Studio spotlight class:
Classname spotlight
color - light color
intensity - light intensity
range - light range
innerconeangle - start falloff angle
outerconeangle - end falloff angle
angles - euler angles for spotlight direction

Torque Engine spotlight class:
Classname light_spot
color - light color
intensity - light intensity
theta - start falloff angle
phi - end falloff angle
falloff2 - light range
target - the name of the target entity the spotlight points at

Directional Lights
A directional light is used to approximate a point light that is infinitely far away. Use these for lighting outdoor scenes. The Torque engine does not have a directional light class.

3D World Studio directional light class:
Classname sunlight
color - light color
intensity - light intensity
angles - euler angles for sunlight direction
