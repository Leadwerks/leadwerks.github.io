Selecting Textures

There are three ways to select textures. You can use the texture category and texture lists in the side panel. You can also select a texture with the texture browser. Finally, you can select a texture from a brush or mesh surface.

You can use the texture category and texture lists in the side panel. Each texture category contains a list of textures for that category. A preview of the texture will be shown in the texture preview display.

The texture browser allows you to browse thumbnails from entire texture categories. Press the Texture Browser toolbar button to open the texture browser.

You can change texture categories by selecting a texture category from the texture browser treeview. Select a texture by clicking the left mouse button on a thumbnail, or by selecting an item in the texture browser treeview. You can view a texture at full size by double-clicking on the thumbnail with the left mouse button.

Close the texture browser window to return to the editor.

You can select textures from a scene by picking the object with the texture you want to make current. Press the Get Texture toolbar button. Select the brush or mesh surface that is painted with the texture you want to make current.
