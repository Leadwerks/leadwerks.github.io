The SMF File Format

September 10, 2006

 

Overview

The .smf file format is a simple model format for fast model loading.

 

Data Types

The following data types are used:

Byte             8-bit unsigned integer

Word            16-bit unsigned integer

Long             32-bit signed integer

Float             32-bit float value

String           null-terminated string

Name            32-bit index to the name table

Color3           24-bit color value

Vec2             2-dimensional float vector

Vec3             3-dimensional float vector

Vec4             4-dimensional float vector

Array            variable length data array

 

Header

Word            mapversion (should be 1)

Byte             mapflags

Long             number of children to the root node

 

The files is then read recursively.  Start with the first child node, and keep going until you get to the last.  For each child, the format looks like this:

 

Vec3   Position

Vec3   Rotation

Vec3   Scale

String  texture group

String  texture name

 

Word   vertex count

 

The vertex count is followed by arrays containing vertex data:

Array   vertex position array (the size is vertex count * 12)

Array   vertex normal array (the size is vertex count * 12)

Array   vertex texcoord array (the size is vertex count * 8)

 

The triangle count is followed by an array containing triangle indice data:

Short  Triangle count

Array   Triangle indice array (the size is vertex count * 6)

 

The limb ends with a value for the number of children this limb has:

Long   Child count

 

Child limbs are listed next.
