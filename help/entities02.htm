Creating Entities

Select the entity object category from the object category list. Select the entity you want to create from the object list.

Click the left mouse button in any of the 2-dimensional viewports. A position marker will appear. You can move this around by clicking and dragging the left mouse button.

Press the right mouse button and select the Create menu. Alternatively, you may press the Enter key.

An entity will be created, in the position you placed the marker. It may take a moment to the first time a mesh is created, because the mesh data has to be loaded for the first time. Subsequent instances of the mesh will be created much quicker.

The entity will be drawn in the editor as a sprite that always faces the camera. In this case, a light bulb image is used.
