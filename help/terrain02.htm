Creating Terrain

Press the terrain editing toolbar button. This will open the terrain editing window.

Press the new button and the new terrain window will appear. The resolution dropdown box chooses how many tiles the terrain is made up of. Once a terrain is created, its resolution cannot be changed. Use 64x64 for small terrains, 128x128 for medium, and 256x256 for large. Smaller resolutions can be used to create small terrains that fit into a brush-based map, like a courtyard with a rolling lawn in the center.

Press OK to create the terrain.

You can load a heightmap from a file. Make sure the terrain tab is selected and press the load button. Choose a heightmap image to load. When the image is loaded, the pixel brightness of each pixel will be used to set the terrain height.
