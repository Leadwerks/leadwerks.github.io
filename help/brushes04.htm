Vertex Moving

Vertex moving allows you to move individual vertices of a solid. Press the Select Vertex toolbar button to switch to select vertex mode. When in select vertex mode you can select solids only by clicking the left mouse button on the solid center cross.

Select a vertex by clicking on it with the left mouse button.

Move the vertex by holding the left mouse button and dragging the mouse.

You can also select and move edges by clicking and dragging the yellow edge tabs.

When moving vertices, care should be taking to preserve correct solid geometry. The resulting solids must be convex and coplanar. Otherwise, errors might results when rendering lights or compiling a BSP.


Vertex moving can create incorrect solid geometry.
