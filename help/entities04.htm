Adding New Entity Classes

You can create custom entity templates to work with your own realtime engine.

Open the file entities.def in a text editor, such as Windows Notepad. You will see a list of light entity classes and properties. You can edit or add to these entities, to create new entity classes that will be available in the editor, from the object list.

@pointclass Light : "sprites\lightbulb"
[
          color(color) : 255 255 255
          intensity(float) : 1.0
          range(integer) : 800
          linearfalloff(choices) : 1
          [
                    0 : false
                    1 : true
          ]
]

@pointclass Spotlight : "sprites\lightbulb"
[
          color(color) : 255 255 255
          intensity(float) : 1.0
          range(integer) : 800
          innerconeangle(integer) : 35
          outerconeangle(integer) : 35
          angles : 90 0 0
]

@pointclass SunLight : "sprites\lightbulb"
[
          color(color) : 255 255 255
          intensity(color) : 255 255 255
          angles : 45 45 0
]
