Options Dialog

The options dialog allows you to adjust editor options like video settings and interface colors.

General Options

  • Instant object creation will create a new solid, model, or entity as soon as you draw it.
  • Select new objects will select a new object as soon as it is created.
  • Draw dashed lines will draw a new solid bounding box with dashed lines instead of solid.
  • Map path is the directory the open map dialog starts in.
  • Import path is the directory the import dialog starts in.
  • Export path is the directory the export dialog starts in.
  • Plugin path is the directory the editor loads plugins from.
  • Model path is the directory the editor scans and loads models from.
  • Texture path is the directory the editor scans and loads textures from.
  • Prefabs path is the directory the editor scans and loads prefabs from.
  • Entity file is the entity definition file (.def) the editor loads entity classes from.

    Video Options

  • Hardware multitexturing will enable single-pass rendering. This should always be enabled.
  • Trilinear filtering will make textures appear less grainy, but will also lower framerate.
  • Anisotropic filtering will make textures appear better, but may lower framerate.
  • Compressed textures will compress texture data in the video memory. This should always be enabled.
  • Sync every frame will make camera movements appear smoother, but will also lower framerate.
  • Show FPS in free-flight mode will display the framerate in the 3D perspective viewport when in free-flight mode.
  • Antialias will make edges appear less jagged, but will also lower framerate.
  • Texture quality can be lowered to reduce the amount of texture data stored in video memory and increase framerate.

    Color Options

    The color options allow you to adjust the colors of various interface items. You can save the current color scheme to a .clr color file, or load a color scheme from a file.
