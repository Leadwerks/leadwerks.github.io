CSG Operations

Solids be used in constructive solid geometry (CSG) operations. This includes carve, hollow, slice plane, and extrude face.

Carve

The carve operation can be used to subtract a volume from a solid. Create two solids, and position them so that one is partly inside the other.

Press the carve toolbar button. The selected solid's volume will be carved out of any solids it intersects. Delete the selected solid to see the carved volume.

The carve operation (and all other CSG operations) preserves correct solid geometry. Rendering in solid mode, we can see that this concave shape is made out of multiple convex solids.

The carve operation can handle very complicated procedures. However, excessive carving is generally not a good idea, as it can create a lot of unnecessary polygons.

Hollow

The hollow operation cuts a volume out of the middle of a solid. Select a solid and press the hollow toolbar button. The hollow dialog will appear. You can adjust the hollow thickness. A negative hollow thickness will cause hollow the solid outwards, increasing its size. Be careful not to use a hollow thickness that exceeds the thickness of the object.

The resulting solid will be hollowed out.

Slice Plane

The slice plane tool allows you to slice a solid in half, along an arbitrary plane. Press the slice plane toolbar button. Click the left mouse button, hold, and drag to begin drawing a line.

Release the mouse button to perform the slice plane operation. All objects that touch the line you drew will be sliced in half.

Extrude Face

The extrude face tool creates a new solid by projecting an existing solid face out and away. Press the select subobject toolbar button to switch to subobject mode. Select a solid face you wish to extrude.

Press the extrude face toolbar button. The extrude face dialog will appear. You can adjust the extrude face thickness.

A new solid will be created, branching off the selected face.

The extrude face tool is especially useful for designing radial features.
