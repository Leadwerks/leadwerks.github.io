Lighting Solids

Solids are lit with lightmaps. They will cast and receive shadows.

Luxel Size

To adjust a solid's lightmap resolution, first press the select subobject toolbar button to switch to select subobject mode. Select the face(s) you wish to alter. Decrease the luxel (lightmap pixel) size for higher-resolution shadows. Increase the luxel size for lower resolution lighting. Use small luxel sizes on surfaces that you want to receive more detailed shadows. Use large luxel sizes on surfaces that the viewer is unlikely to see up close, like the top of a roof, or on surfaces on which no shadows will be cast.

Lightmap Leaks

Sometimes, a low-resolution lightmap will display lightmap leaks, spots where the light "leaks" through a solid wall.


Light can "leak" through solid objects when using a low-resolution lightmap.

Lightmap leaks can be minimized by using a higher resolution lightmap.

Beveled joints can also minimize lightmaps leaks. This has the additional benefit of decreasing polycount, because the joint faces can be removed.
