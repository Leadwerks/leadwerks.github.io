Creating Solids

Select the Solids category from the Object Category list. You can choose several basic shapes to create. Some solids, like the cylinder, have parameters that will change the way a solid is created.

Click and draw the left mouse button in a 2-dimensional viewport. A drawing box outline will be created.

Press the right mouse button and select the Create menu. Alternatively, you may press the Enter key.

A solid will be created, filling the dimensions of the drawing box.

More complex shapes are available in the Extended Primitives object category. Note that although shapes such as a tube are concave, they are made up of multiple convex solids.
